# UI Regression Checklist

Use this list to validate UI-only updates without touching business logic.

## Auth
- Login renders and submits correctly
- Error state displays clearly
- Show/Hide password works

## Dashboard
- KPIs render and values load
- Filters apply as before
- Follow-up actions open modals and save correctly
- Stuck leads widget renders

## Leads
- List loads, filters apply, chips update
- Action icons visible on mobile
- Detail page loads, timeline renders
- Update status flow works

## Companies
- List loads and filters apply
- Detail page renders and contact links work

## Contacts
- List loads, filters apply
- Modal opens and saves
- Email/phone links are clickable

## Proposals
- List loads, filters apply
- Details page renders, actions work
- Editor toolbar works (bold, lists, tables)
- Preview loads and paginates

## Tasks
- List loads, filters apply
- Complete/edit/delete actions work

## Reports
- Report loads and table renders
- Export buttons trigger download

## Analytics
- Cards and sections render

## Users/Settings
- Users list loads, filters apply
- Settings tabs switch correctly
- Forms submit as before
